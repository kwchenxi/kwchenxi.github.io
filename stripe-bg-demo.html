斜条纹背景效果演示

CSS 代码说明： 使用 linear-gradient 创建 45 度角的斜条纹，条纹颜色为浅灰色 (#f0f0f0)，与透明色交替，background-size 控制条纹大小。

推荐使用的CSS代码

.stripe-bg { /* 5px粗细的斜条纹背景，45度角，浅灰色条纹 */ background: linear-gradient(45deg, #f0f0f0 15%, transparent 15%, transparent 50%, #f0f0f0 50%, #f0f0f0 65%, transparent 65%, transparent); background-size: 5px 5px; }

使用建议： 可以通过调整 background-size 的值来改变条纹的粗细，调整 linear-gradient 中的颜色值来改变条纹颜色，调整角度（如 30deg, 60deg）来改变条纹方向。
